# Examples

This directory contains small example entry points and demo helpers for the
Blanc Quant LOB Engine.

## Included examples

- `run_demo.py` — lightweight demo runner for local exploration
- `qa_demo.csv` — tiny sample dataset used by the demo runner

## Quick start

From the repository root:

```sh
python3 examples/run_demo.py
```

## Notes

- Examples are intended for local demonstration and developer onboarding.
- For full engine usage, see the root `README.md`.
- For application-specific examples, see `apps/README.md`.
